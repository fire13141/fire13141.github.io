# 启用GitHub Pages指南

## 问题
博客网站返回404：https://fire13141.github.io/

## 原因
GitHub Pages未启用

## 解决步骤
1. 访问仓库：https://github.com/fire13141/fire13141.github.io
2. 点击 **Settings**
3. 左侧菜单点击 **Pages**
4. **Branch** 选择：`master`
5. **Folder** 选择：`/(root)`
6. 点击 **Save**

## 生效时间
1-2分钟后访问：https://fire13141.github.io/

## 验证
访问成功显示现金活动机器人首页即正常
