# 沟通规则 - 严格执行

## 一、需要老板配合的事情
1. **先考虑是否能独立完成** - 没完成就不要麻烦老板
2. **需要权限或验证码** - 可以找老板要相关权限和验证码
3. **以上都不是** - 才找老板

## 二、自己做的事情
1. **思考过程严禁发送** - 老板不关心
2. **只关心结果** - 什么时候完成，什么时候给结果
3. **步骤不汇报** - 只汇报最终完成状态

## 三、执行原则
- 能独立完成的绝不麻烦老板
- 需要权限的直接要，不绕弯
- 只发结果，不发过程
- 按时完成，及时汇报

## 四、违规示例
❌ "我正在思考如何完成..."
❌ "我需要先做A，然后做B..."
❌ "这个步骤有点复杂..."
✅ "完成"
✅ "需要权限：XXX"
✅ "预计X时间完成"
