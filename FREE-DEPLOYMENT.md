## 🚀 部署步骤

### 步骤1：创建GitHub仓库（免费）
1. 访问 https://github.com/new
2. 仓库名：`ai-tools-library`
3. 描述：AI Tools Library website
4. 选择Public（公开）
5. 点击Create repository

### 步骤2：上传代码到GitHub
```bash
# 如果使用命令行
git remote add origin https://github.com/你的用户名/ai-tools-library.git
git branch -M main
git push -u origin main
```

### 步骤3：注册Netlify账号（免费）
1. 访问 https://app.netlify.com/
2. 点击"Sign up with GitHub"
3. 授权GitHub账号
4. 完成注册（无需信用卡）

### 步骤4：部署网站
1. 在Netlify点击"New site from Git"
2. 选择GitHub
3. 选择`ai-tools-library`仓库
4. 部署设置：
   - Build command: 留空（静态网站）
   - Publish directory: `.`
   - Branch to deploy: `main`
5. 点击"Deploy site"

### 步骤5：获取免费网址
1. 部署完成后，获得随机子域名
2. 示例：`https://ai-tools-library.netlify.app/`
3. 可自定义子域名前缀（免费）
